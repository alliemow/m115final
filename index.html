My Favourite Cloud Based Apps
Alex Dinning

Google Docs

https://drive.google.com - Google Docs
- Used to write,create and edit documents whereever you are
- Basic and Easy to use, with many different templates

Prezi

https://prezi.com
- Used to Make presentations - Uses multiple paths linked together to create an unique slide show

Drop box

https://www.dropbox.com/
- Gives you 2gb for free before paying
- Used to Store pics,documents, etc. over a range of devices

Google Play Music

https://play.google.com/music/listen
- Used to play music
- Can download music and play from anywhere using the cloud storage

Evernote

https://evernote.com
- Used to maintain and organize
- Can Write notes, checklists,research, organize web articles, docs, and photos, while communicating to people with built in chat

KeepSafe

https://www.getkeepsafe.com/
- Used to keep personal stuff secure and protected
- uses sucurity lock sequence(pin), with also a back up

WhatsApp

https://www.whatsapp.com/
- Users are allowed to exchange messanges without having to pay for SMS(Uses internet connection)
- Allows users to store all messages(text,pictures,etc.) online in cloud service.

These are the apps I found most
useful to me.I hope you enjoyed
